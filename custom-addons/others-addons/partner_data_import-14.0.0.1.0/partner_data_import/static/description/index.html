Import Partner Data

Import Partner Data from CSV or XLS with Different columns.

This module has been tested in Community and Enterprise Version 14.


Highlights Of The Application

- Import partner Data from CSV or XLS with different options.

- Feature to create new partner or update existing partner.

- Feature to import data using background process.

- Feature to notify user when import process done.

- Added History Table To Track import process time, failed rows etc.


Complete process for configuration and functionality explained below.

After installation, go to Settings, here you will see one menu Import Partner Data. By using this menu, you can import partner and also can see imported partner track record




To import partner data, click on Import Partner Data >> Import, on click Import , you will see one wizard will open. Here you can upload file and configure different options




Fill details and click on Import button to import. When you click on Import button it will create one track record to check import process progress.




As import process run in background, so when process over. It will automatically notify to user to check result. Also it create history track record to analyse import process. it includes import time duration, failed rows track record etc.
